Earthquake Classifier and Map Visualizer

Project Overview

This project fetches recent earthquake data from GeoNet, classifies earthquakes based on their potential intensity (High or Low) using a Decision Tree model, and visualizes the results on an interactive map.

Data Source: Earthquake data is provided by GeoNet (https://www.geonet.org.nz/).

Note: This project serves as a preliminary attempt to learn and demonstrate the application of Decision Trees for a simple classification task.

How to Use

To generate the earthquake map and run the classifier:

  1. Ensure you have Python and the necessary libraries installed (see Technologies Used).
  2. Run the main script from your terminal:

    3. python quake_classifier.py

  3. This will fetch data, process it, and generate an HTML map file.
  4. Open the generated quake_map.html file in your web browser to view the map.

Configuration

You can modify the behavior of the script by editing the config.py file. Key settings include:

  • DAYS_FILTER: Number of past days to fetch earthquake data for.
  • MIN_MAGNITUDE: Minimum magnitude of earthquakes to include.
  • INTENSITY_THRESHOLD: Magnitude threshold used to define 'High' vs 'Low' intensity for actual intensity calculation and decision tree training.
  • MAP_INTENSITY_TYPE: Set to "predicted" to show intensities predicted by the Decision Tree, or any other value (e.g., "actual") to show intensities based on the INTENSITY_THRESHOLD.

Technologies Used

  • Python
  • Pandas (for data manipulation)
  • Scikit-learn (for the Decision Tree Classifier)
  • Matplotlib (for plotting the decision tree)
  • Leaflet (JavaScript library for interactive maps)
